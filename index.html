Installation

How to upload a .instapage file

1. Don’t have an instapage account yet? Sign up for a FREE account right now. If you do have one, sign in at https://app.instapage.com/auth/login

2. From the main dashboard click Create New page

3. On the next page select Upload .instapage File

4. Click “Choose File” and find the .instapage file you’d like to upload on your hard drive.

5. Your page should now be uploaded as a new page onto your main Dashboard screen.
